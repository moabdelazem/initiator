# Initiator

A CLI tool to scaffold and manage Go project structures",
Long: `Initiator is a powerful command-line tool designed to simplify Go development.
It helps developers generate scalable Go project structures with pre-defined templates
and integrates seamlessly with popular tools and frameworks. Whether you're starting
a new web service, CLI app, or library, Initiator provides the essential scaffolding
and features to get started quickly.

## Try

```bash
git clone https://github.com/moabdelazem/initiator.git
make build
./bin/initiator create github.com/user/project_name --path ~/project_path
```
